Is IP used in Microsoft Azure?

Microsoft maintains a public list of IP addresses used by the Azure datacenters. Instead of downloading the list to see if a particular IP is an Azure IP you can use this site.

Which region is the IP used in?

The public list also contains information about which region that the IP is used in.

Is a site hosted in Azure?

By combining the DNS record for a domain together with Microsofts public Azure datacenter IPs we can determine if a site is hosted in Azure.

Which region is the site hosted in?

When the IP of a site is found, we can use the datacenter IPs to determine which data center the site is hosted (note this is not 100% reliable if e.g. trafic manager is used, or other multi-region load-balancing)
